Html Interview Question & Answer:

1. What is HTML and why is it important? +
HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content using elements like headings, paragraphs, images, and links. It is essential because it forms the backbone of web development.
2. What are the key differences between HTML4 and HTML5? +
HTML5 introduced new elements like <article>, <section>, <video>, and <audio>, improved form controls, and removed outdated elements from HTML4.
3. What is the purpose of the <head> and <body> sections in an HTML document? +
The <head> section contains metadata, styles, and scripts, while the <body> section contains the actual content that appears on the webpage.
4. What are semantic HTML elements and why should we use them? +
Semantic elements like <header>, <nav>, <article>, and <footer> provide meaning to web content, improving accessibility, SEO, and code readability.
5. How does HTML work with CSS and JavaScript? +
HTML structures the webpage, CSS styles it, and JavaScript adds interactivity, creating a complete and dynamic web experience.
6. What is the difference between <div> and <span> tags? +
<div> is a block-level element used for grouping larger sections, while <span> is an inline element for styling parts of text within a line.
7. How can I make my HTML page responsive? +
Use CSS media queries, frameworks like Bootstrap, and the viewport meta tag: <meta name="viewport" content="width=device-width, initial-scale=1">.
8. What is the difference between HTML and XHTML? +
HTML is more lenient with syntax, while XHTML is stricter and follows XML rules, requiring tags to be properly nested, lowercase, and closed.
9. What is the purpose of the <meta> tag in HTML? +
The <meta> tag provides metadata about the HTML document, such as author, description, keywords, and viewport settings, which help search engines and browsers understand the page content.
10. What is the difference between <section> and <div> tags in HTML? +
The <section> tag represents a thematic grouping of content with a heading, while <div> is a generic container for styling and layout purposes.
11. What is the use of the "alt" attribute in images? +
The "alt" attribute provides alternative text for an image, which is displayed if the image cannot be loaded. It also helps with accessibility and SEO.
12. How can you create a hyperlink that opens in a new tab? +
Use the target attribute with "_blank": <a href="https://example.com" target="_blank">Visit Example</a>.
13. What is the difference between inline, block, and inline-block elements? +
Inline elements do not start on a new line and take up only necessary width. Block elements start on a new line and occupy full width. Inline-block elements behave like inline elements but allow setting width and height.
14. What is the purpose of the "action" and "method" attributes in HTML forms? +
The "action" attribute defines the URL to which the form data is sent, while the "method" attribute specifies how the data is sent: "GET" appends data to the URL, "POST" sends data in the request body.
15. What is the difference between <em> and <strong> tags in HTML? +
The <em> tag emphasizes text with italics, while the <strong> tag indicates strong importance, usually shown in bold.
